# Network
Opis repozytorium
Repozytorium zawiera projekt systemu sieciowego wykonany w środowisku symulacyjnym Cisco Packet Tracer. Celem projektu jest zaprojektowanie, konfiguracja oraz analiza działania sieci komputerowej z wykorzystaniem podstawowych i wybranych zaawansowanych technologii sieciowych.
W ramach projektu została utworzona topologia sieci, skonfigurowano adresację IP oraz przeprowadzono subnetting. Zaimplementowano podstawowe usługi sieciowe, takie jak DHCP oraz DNS, a także skonfigurowano routing (statyczny i/lub dynamiczny) oraz sieci VLAN.
Repozytorium zawiera pliki konfiguracyjne, plik projektu (.pkt) oraz dokumentację przedstawiającą etapy realizacji, testy działania sieci oraz proces diagnostyki i optymalizacji.
Projekt ma na celu rozwijanie praktycznych umiejętności w zakresie projektowania i zarządzania sieciami komputerowymi.
